# Guild Village Helper addon 3.3.5a

- Guild Village Helper je addon pro modul Guild Village.  
- Zobrazuje důležité informace pro Guild Village a umožňuje pohodlnější teleport bez nutnosti psát příkazy.  
 
##
- Guild Village Helper is an add-on for the Guild Village module.  
- It displays important information for Guild Village and allows for more convenient teleporting without having to type commands.
